# フロントエンド

React + TypeScript による SPA フロントエンド

## 技術スタック

- React 18 + TypeScript + Vite
- shadcn/ui + Tailwind CSS
- React Query
- React Hook Form + Zod
- Supabase Auth

## セットアップ

```bash
npm install
cp .env.example .env.local
npm run dev
```

## 主要機能

- 認証（メール/パスワード、Google OAuth）
- 企業管理
- 予定管理
- カレンダー表示
